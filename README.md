# 在庫管理アプリ

スマホ操作前提・ミニマル設計の在庫管理アプリです。

- **画面**: GitHub Pages（静的サイト・無料）
- **API**: Google Apps Script（無料）
- **データ**: Google スプレッドシート

```
[スマホ] → GitHub Pages（画面） → Apps Script（API） → スプレッドシート（保存）
```

## 機能
- 在庫一覧（保管場所タブ・カテゴリ・商品名検索で絞り込み）
- 入庫 / 出庫（±1のクイック操作 と 数量指定）
- 商品の新規追加 / 編集 / 削除
- 拠点間の在庫移動（学大・笹塚・田村・Graz）
- 在庫切れアラート（しきい値を下回ると赤く強調）
- 共有パスワードによるログイン（パスワードは端末に保存。コードには含めない）

## セットアップ手順

### 1. スプレッドシートを用意
1. [Google スプレッドシート](https://sheets.new) で新規ファイルを作成
2. ファイル名は任意（例: 在庫管理データ）

### 2. Apps Script を設定
1. スプレッドシートのメニュー **拡張機能 → Apps Script** を開く
2. 既存コードを全部消し、`gas/Code.gs` の中身を貼り付け
3. 先頭の `const PASSWORD = 'CHANGE_ME';` を**自分たちのパスワードに変更**
4. 保存（💾）

### 3. ウェブアプリとしてデプロイ
1. Apps Script 右上の **デプロイ → 新しいデプロイ**
2. 種類の選択（⚙）→ **ウェブアプリ**
3. 設定:
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**
4. **デプロイ** → 初回は権限承認（自分のGoogleアカウントで許可）
5. 表示される **ウェブアプリURL**（`https://script.google.com/macros/s/.../exec`）をコピー
   - ブラウザでそのURLを開き「稼働中です」と出れば成功

> ⚠️ コードを修正したら **デプロイ → デプロイを管理 → 編集（鉛筆）→ バージョン: 新バージョン → デプロイ** で反映してください。

### 4. フロントに URL を設定
`js/config.js` を開き、コピーしたURLを貼り付け:
```js
const CONFIG = {
  GAS_URL: 'https://script.google.com/macros/s/XXXX/exec'
};
```

### 5. GitHub Pages で公開
1. このフォルダを GitHub リポジトリにpush
2. GitHub の **Settings → Pages**
3. Source: **Deploy from a branch** / Branch: `main`（または `master`）/ フォルダ: `/ (root)`
4. 数分後 `https://ユーザー名.github.io/リポジトリ名/` で公開

### 6. スマホで使う
- 公開URLをスマホで開く → パスワード入力でログイン
- Safari/Chrome の「ホーム画面に追加」でアプリのように使えます（PWA対応）

## データ構造（スプレッドシート「在庫」シート）
| 列 | 内容 |
|----|------|
| id | 自動採番（編集不要） |
| 商品名 | |
| カテゴリ | 抹茶 / ほうじ茶パウダー / 煎茶 / ほうじ茶 / 和紅茶 / 玄米茶 |
| 保管場所 | 学大 / 笹塚 / 田村 / Graz |
| 在庫数量 | |
| 単位 | 個 / g / kg / 本 / 袋 / 箱 |
| 原価 | 円 |
| 仕入元 | どこから仕入れたか（商品マスタから自動入力） |
| しきい値 | この数以下で在庫切れアラート。新規追加時は登録数量の15%が自動設定される（編集で変更可） |
| 更新日時 | 自動 |

シートは初回アクセス時に自動生成されます。直接編集も可能です。
列構成を変更したときは、シート1行目のヘッダーが自動で更新されます。

## 商品マスタ（「商品マスタ」シート）
商品名の表記ゆれを防ぐため、商品の一覧を別シートで管理します。
| 列 | 内容 |
|----|------|
| 商品名 | 一意（重複させない） |
| カテゴリ | 既定値 |
| 単位 | 既定値 |
| 標準原価 | 既定値（円） |
| 仕入元 | 既定値（どこから仕入れたか） |

- 在庫の新規追加画面では、商品名をこのマスタから**プルダウンで選択**します。
  選ぶとカテゴリ・単位・原価が自動入力されます。
- リストに無い商品は「**＋ 新しい商品を登録**」を選んで入力すれば、
  保存時にこのマスタへ**自動追加**されます。
- 既定値の修正や不要商品の削除は、この「商品マスタ」シートを直接編集してください。

## 選択肢を変えたいとき
`gas/Code.gs` 冒頭の定数を編集して再デプロイすれば、画面の選択肢も自動で変わります。
- 保管場所: `LOCATIONS`
- カテゴリ: `CATEGORIES`
- 単位: `UNITS`

## セキュリティについて
身内・社内数人での利用を想定した「簡易パスワード保護」です。
- パスワードはコード/リポジトリには含めず、ログイン時に各端末へ保存します
- 本格的な権限管理（個人別ログイン等）が必要になったら拡張できます
